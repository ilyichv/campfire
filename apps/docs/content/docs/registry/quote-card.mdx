## Installation

<Tabs items={["CLI", "Manual"]}>

<Tab value="CLI">

```bash

```

Or with any shadcn-compatible CLI:

```bash
bunx shadcn@latest add https://campfire.dev/r/quote-card.json
```

## Usage

Pairs well with the [Quote layout](/docs/registry/quote), which centers a
single quotation on the slide:

```mdx title="slides/04-quote.mdx"
---
layout: quote
title: What people say
---

<QuoteCard
  quote="The deck that lives in git is the deck that stays true."
  author="Ada Lovelace"
  role="Analyst, Analytical Engines"
/>
```
