## Installation

<Tabs items={["CLI", "Manual"]}>

<Tab value="CLI">

```bash
camp add quote-card
```

Or with any shadcn-compatible CLI:

```bash
bunx shadcn@latest add https://campfire-deck.vercel.app/r/quote-card.json
```

</Tab>

<Tab value="Manual">

Copy the source into `components/quote-card.tsx`. It imports `cn` from
`@/lib/utils` — scaffolded by `camp init`, or `camp add utils`:

<ComponentSource name="quote-card" title="components/quote-card.tsx" />

</Tab>

</Tabs>

## Usage

Pairs well with the [Statement layout](/docs/registry/statement), which
centers a single big idea on the slide:

```mdx title="slides/04-quote.mdx"
---
layout: statement
---

<QuoteCard
  quote="The deck that lives in git is the deck that stays true."
  author="Ada Lovelace"
  affiliation="Analyst, Analytical Engines"
/>
```

## API Reference

Extends `ComponentProps<"figure">` — native props are forwarded and
`className` is merged last. Parts expose `data-slot="quote-card"`,
`quote-card-quote`, and `quote-card-attribution`.
