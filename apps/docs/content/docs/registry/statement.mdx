## Usage

The slide body renders at display scale, centered. `title` becomes a small
eyebrow above it. It is also the natural home for a
[Quote Card](/docs/registry/quote-card):

```mdx title="slides/04-quote.mdx"
---
layout: statement
---

<QuoteCard
  quote="The deck that lives in git is the deck that stays true."
  author="Ada Lovelace"
/>
```

## API Reference

`data-slot="statement-layout"` and `statement-layout-statement`.
