## Installation

<Tabs items={["CLI", "Manual"]}>

<Tab value="CLI">

```bash
camp add section
```

Or with any shadcn-compatible CLI:

```bash
bunx shadcn@latest add https://campfire.dev/r/section.json
```

</Tab>

<Tab value="Manual">

Copy the source into `layouts/section.tsx`. It imports `cn` from
`@/lib/utils` — scaffolded by `camp init`, or `camp add utils`:

<ComponentSource name="section" title="layouts/section.tsx" />

</Tab>

## Usage

Works with an empty body too — a bare divider between chapters:

```mdx title="slides/04-numbers.mdx"
---
layout: section
title: Numbers
---
```

The background uses the theme's `--color-primary`; change the theme and every
section divider follows.
