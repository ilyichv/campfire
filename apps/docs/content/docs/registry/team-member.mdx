## Usage

Without `src` the avatar falls back to the person's initials. Photos go in
`assets/` (served from `/`):

```mdx title="slides/07-team.mdx"
---
title: Team
---

## The team

<TeamMember name="Ada Lovelace" position="CEO" src="/team/ada.jpg" />
<TeamMember name="Grace Hopper" position="CTO" />
```

## API Reference

Extends `ComponentProps<"figure">` — native props are forwarded and
`className` is merged last. Parts expose `data-slot="team-member"`,
`team-member-photo`, `team-member-name`, `team-member-position`, and
`team-member-detail`.
