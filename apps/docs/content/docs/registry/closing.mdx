## Installation

<Tabs items={["CLI", "Manual"]}>

<Tab value="CLI">

```bash
camp add closing
```

Or with any shadcn-compatible CLI:

```bash
bunx shadcn@latest add https://campfire.dev/r/closing.json
```

</Tab>

<Tab value="Manual">

Copy the source into `layouts/closing.tsx`. It imports `cn` from
`@/lib/utils` — scaffolded by `camp init`, or `camp add utils`:

</Tab>

</Tabs>
