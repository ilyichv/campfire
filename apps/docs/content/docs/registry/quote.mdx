## Installation

Installs the layout and pulls in [Quote Card](/docs/registry/quote-card) via
`registryDependencies`.

<Tabs items={["CLI", "Manual"]}>

<Tab value="CLI">

```bash
camp add quote
```

Or with any shadcn-compatible CLI:

```bash
bunx shadcn@latest add https://campfire.dev/r/quote.json
```

</Tab>

<Tab value="Manual">

Copy the source into `layouts/quote.tsx`, then install
[Quote Card](/docs/registry/quote-card):

<ComponentSource name="quote" title="layouts/quote.tsx" />

</Tab>

</Tabs>

## Usage

The layout only centers its children — pair it with `QuoteCard` for the
typography:

```mdx title="slides/05-quote.mdx"
---
layout: quote
---

<QuoteCard quote="…" author="Grace Hopper" />
```
